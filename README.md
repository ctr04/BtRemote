## <div align="center">Touchpad for Android Desktop Mode</div>

<div align="center"><h4>An Android app that turns your smartphone into a Touchpad for Android 16's Desktop Mode.</h4></div>

[![License: GPLv3](https://img.shields.io/badge/License-GPLv3-blue)](https://www.gnu.org/licenses/gpl-3.0)

## Description

This is a fork of [BT Remote](https://gitlab.com/Atharok/BtRemote) in very early development. Turn your smartphone into a Touchpad to navigate and interact with Android 16's Desktop Mode.

Features:

- Developed with Kotlin, Jetpack Compose, and Material 3.
- Privacy-focused with no data collection (internet permission disabled).
- No advertisements.
- Open source.
